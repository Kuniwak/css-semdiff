CSS Semantic Diff
=================

Install
-------

```shell
npm install -g git@github.lo.mixi.jp:Kuniwak/css-semdiff
```


Usage
-----

### Comparing by Abstract Syntax Tree

```shell
css-astdiff a.css b.css
```


#### Options

- `--verbose`: Display verbose output


### Comparing by Rule Order

```shell
css-orderdiff a.css b.css
```


#### Options

- `--verbose`: Display verbose output
